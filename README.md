# Andrew Byrley - Personal Website

A personal portfolio website for an AI Governance & Policy expert, designed to reflect a "Diplomat & Engineer" aesthetic.

## features

-   **Clean, Professional Design**: "Diplomat & Engineer" theme using a Navy color palette.
-   **Responsive Layout**: Mobile-friendly navigation and flexible grid for projects/career timeline.
-   **Press Kit**: Dedicated `/press.html` page with downloadable high-res headshots and copy-paste bios.
-   **No Frameworks**: Built with semantic HTML5, modern CSS3 (Variables, Flexbox), and Vanilla JavaScript.

## tech stack

-   **HTML5**
-   **CSS3** (Custom properties for theming)
-   **JavaScript** (ES6+)

## local development

No build process required. To run locally:

```bash
# Using Python 3
python3 -m http.server 8080
```

Open `http://localhost:8080` in your browser.
